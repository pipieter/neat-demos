# Neat demos

Collection of demos using [neat headers libraries](https://github.com/pipieter/neat-headers).

| Demo   | Description          | Headers                                                                        | External libraries                                                                                                                          |
| ------ | -------------------- | ------------------------------------------------------------------------------ | ------------------------------------------------------------------------------------------------------------------------------------------- |
| ball3d | 3D ball-in-maze game | [ecs](https://github.com/pipieter/neat-headers/blob/main/include/neat/ecs.hpp) | [raylib](https://github.com/raysan5/raylib), [R3D](https://github.com/Bigfoot71/r3d), [Jolt Physics](https://github.com/jrouwe/JoltPhysics) |
